### tag映射规则
1. 造成伤害 -> 伤害
2. 埋葬友方/敌方或是友方被埋葬、卡片被埋葬 -> 友方/敌方埋葬
3. 埋葬后X卡 -> 随机埋葬
4. deathrattle -> 亡语
5. 萦绕/linger -> 萦绕
6. 强化、置顶敌方【诅咒】、消耗敌方【诅咒】力量、当敌方【诅咒】揭晓时、转移力量到敌方【诅咒】-> 污染
7. 生成【次元裂缝】-> 任意门
8. 去除【次元裂缝】-> 吞噬
9. 置顶友方（包括自身） -> 预言
10. 给予友方【力量】-> 强化
11. 消耗、转移【力量】-> 消化

### tag正负分类
- 伤害：正面
- 埋葬友方：负面
- 埋葬敌方：正面
- 随机埋葬：正面
- 亡语：负面
- 萦绕：负面
- 污染：正面
- 任意门
- 吞噬
- 预言
- 强化
- 消化
